#LogIn System: 
---------------
1. Login component ---------> user_name, passwd 
                    ----------> authSlice :
                        --------->api send to backend ---------------> received token ------------> save to localStorage with time as cookies (client_side storage) 
                        ------------> write=> fn: check the token is valid or not 
                        ------------> **** in the Navbar component import the check_token fn for checking , token valid or not, if not valid redirect to login page. 

-------------------------------------------------------- best flow --------------------------------------------

┌───────────────────────────┐
│        Login Page          │
│   (Login Component UI)     │
└──────────────┬────────────┘
               │ user enters
               │ username & password
               ▼
        ┌────────────────┐
        │  authSlice.js   │
        │ (Redux Toolkit) │
        └───────┬────────┘
                │ dispatch login()
                ▼
        ┌──────────────────────┐
        │   Call Backend API    │
        │  POST /api/login      │
        └────────┬─────────────┘
                 │ credentials sent
                 ▼
          ┌─────────────────┐
          │  Backend Server │
          └────┬────────────┘
               │ returns token (JWT)
               ▼
        ┌─────────────────────────────┐
        │  Store Token in Client Side │
        │ localStorage + expiry time  │
        └─────────┬──────────────────┘
                  │
                  ▼
       ┌─────────────────────┐
       │ checkToken() method │
       │ (inside authSlice)  │
       └─────────┬──────────┘
                 │ verifies:
                 │ - token exists?
                 │ - token expired?
                 ▼
     ┌───────────────────────────┐
     │  Navbar Component Mounts   │
     └──────────────┬────────────┘
                    │ imports checkToken()
                    ▼
         ┌──────────────────────────┐
         │  checkToken() executes    │
         └────────────┬─────────────┘
                      │ valid token?
              ┌───────┴─────────┐
              │                 │
              ▼                 ▼
     ┌─────────────────┐   ┌─────────────────────────┐
     │ Allow Navigation │   │ Redirect to Login Page  │
     │ (User Auth OK)   │   │ remove token (optional) │
     └─────────────────┘   └─────────────────────────┘
